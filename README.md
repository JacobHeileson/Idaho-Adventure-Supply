# Idaho Adventure Supply

Idaho Adventure Supply is a responsive Bootstrap 5 website created for a Web Development final project. The website showcases outdoor adventure gear using modern Bootstrap components and responsive web design principles.

## Features

- Responsive navigation bar
- Bootstrap carousel slideshow
- Product showcase cards
- Contact form
- Mobile-friendly responsive layout
- Bootstrap 5 styling and components

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5

## Purpose

This project was created to demonstrate the use of Bootstrap components, responsive design, and structured webpage layout for an Intermediate Web Development course.
